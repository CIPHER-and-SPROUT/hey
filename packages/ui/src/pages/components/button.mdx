# Button

Buttons are to be used as control that triggers an action. The content of the button has to express the action of the button itself.

### Sizes

<Container>
  <Button size="sm">Login</Button>
  <Button size="md">Login</Button>
  <Button size="lg">Login</Button>
</Container>

</Container>
